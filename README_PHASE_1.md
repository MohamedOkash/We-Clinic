## 🎯 حالات الاستخدام المدعومة

### السيناريو 1: طلب تحليل بسيط
```
1. 👨‍⚕️ الطبيب يختار مريض
2. 👨‍⚕️ يذهب لـ "Lab Orders"
3. 👨‍⚕️ ينشئ طلب CBC
4. 🔔 المعمل يتلقى إشعار
5. 🔬 المعمل يرفع النتائج
6. 🔔 الطبيب يتلقى إشعار
7. 👨‍⚕️ يرى النتائج في السجل الطبي
```

### السيناريو 2: طلب أشعة عاجل
```
1. 👨‍⚕️ الطبيب ينشئ طلب X-Ray
2. 👨‍⚕️ اختار الأولوية: "عاجل" 🔴
3. 🔔 إشعار حمراء للمعمل
4. 🔬 المعمل يبدأ الفحص فوراً
5. 🔬 يرفع الصورة والتقرير
6. 🔔 إشعار أخضر: "نتيجة جاهزة"
7. 👨‍⚕️ الطبيب يعرض الأشعة مباشرة
```

### السيناريو 3: تتبع السجل الطبي
```
1. 👨‍⚕️ الطبيب يختار مريض
2. 👨‍⚕️ يضغط "Medical History"
3. 📊 يرى الإحصائيات الشاملة
4. 📝 يرى سجل الزيارات
5. 💊 يرى آخر الروشتات
6. 🔬 يرى آخر التحاليل
7. 🧡 يرى الحساسية والأمراض المزمنة
```
